# Travel Controller 🚌

Este é um projeto desenvolvido em **C++** para simular rotas, viagens e escalas de transporte de passageiros.

## 🚀 Principais Funcionalidades
* Cálculo de rotas otimizadas utilizando o **Algoritmo de Dijkstra**.
* Gerenciamento do tempo e de escalas usando **Listas Encadeadas** na classe `Viagem`.
* Sistema de fadiga e descanso de motoristas.
* Persistência de dados em arquivos `.txt`.

## 🛠️ Como compilar e executar
No terminal do Linux, utilize os comandos:
1. `make main` (para compilar)
2. `./main` (para iniciar)
